An extension that enables dark mode for Google Classroom using custom CSS based on Material Design.

## Supported pages

Pages on Google Classroom that work with Classroom Dark Mode

### Student

- [x] Home
- [x] Your work
- [x] Calendar
- [x] To-do
- [x] Archived classes
- [x] Settings
- [x] Stream
- [x] Classwork
- [x] People

### Teacher

- [x] Home
- [x] Calendar
- [x] To review
- [x] Archived classes
- [x] Settings
- [x] Stream
- [x] Classwork
- [x] People
- [x] Grades
- [x] Class code
- [x] Customize appearance
- [x] Student work
- [x] Class settings
- [x] Review work
